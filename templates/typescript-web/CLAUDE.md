# Project: [プロジェクト名]

TypeScript + React の Web アプリケーション。

## Commands
- `pnpm dev` - 開発サーバー
- `pnpm build` - ビルド
- `pnpm test` - テスト
- `pnpm lint` - リント

## Code Style
- ES modules (import/export)
- 関数コンポーネント + Hooks
- 型定義は必須

## Architecture
- `src/components/` - UI コンポーネント
- `src/hooks/` - カスタムフック
- `src/lib/` - ユーティリティ
- `src/types/` - 型定義
